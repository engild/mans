## sort命令

对文本排序

### 用法
```
sort [OPTION]... [FILE]...

sort [OPTION]... --files0-from=F
```

| 选项  | 说明                                                         |
| ----- | ----------------------------------------------------------- |
|-b     | 忽略每一行开头的空格      |
|-d     | 只考虑空格、数字和字母        |
|-f     | 将小写字母当成大写字母看待        |
|-u     | 去重      |
|-r     | 反向      |
