## grep命令

### 用法
```
grep [OPTION]... PATTERN [FILE]...
```

| 选项 | 说明                     |
| ---- | ------------------------ |
| -i   | 忽略大小写               |
| -E   | 扩展正则表达式           |
| -w   | 强制<模式>仅完全匹配字词 |
| -x   | 强制<模式>仅完全匹配整行 |
| -r   | 递归                     |

杂项：

| -s   | 不显示错误信息       |
| ---- | -------------------- |
| -v   | 选中不匹配的行。取反 |

输出控制：

| -m <次数> | 匹配多少次后停止       |
| --------- | ---------------------- |
| -n        | 打印行号               |
| -H        | 为输出行打印文件名     |
| -h        | 输出时不显示文件名前缀 |
| -q        | 不显示所有常规输出     |

文件控制：

| -A NUM              | 打印文本及其后面NUM 行 |
| ------------------- | ---------------------- |
| -B NUM              | 打印文本及其前面NUM 行 |
| -C NUM 或者直接-NUM | 打印文本及其前后NUM 行 |

### 示例

~~~shell
# 以‘或’的关系匹配多个条件
grep -E 'x|y'
egrep 'x|y'
grep -e x -e y

# 匹配以#开头，以;结尾
grep '^#.*;$'

# 匹配空行
grep ^$
~~~
