## kubectl config命令

### 用法
```
kubectl config SUBCOMMAND [options]
```

### 子命令

| current-context | 显示当前context                     |
| --------------- | ----------------------------------- |
| get-contexts    | 显示config中所有context             |
| use-context     | 设置 kubeconfig 文件中的当前context |
|                 |                                     |
|                 |                                     |
|                 |                                     |
